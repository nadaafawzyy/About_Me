# About Me

A simple “About Me” personal page built with HTML and CSS.  
This project showcases a brief introduction about myself and links to my socials.

## 📌 Overview

This is a static personal webpage where visitors can:

- See a brief introduction about me
- View my profile details, photo, and bio
- Navigate to my social links
- Explore an introductory personal site layout

> ⚠️ **Note:** The **resume** and **portfolio sections** are currently *added as likes* in navigation, but they are **not available yet** (work in progress).

## 🧠 Built With

- HTML5
- CSS3
